Welcome!

Demoshell is a rapid prototyping environment built on the AribaWeb framework. Demoshell supports writing the code for AWComponents (normally contained .java files sitting along side their corresponding .awl template) instead as server-side JavaScript or Groovy script embedded in server (JS) or groovy tags right inside your .htm template. (Yes, mixing code and HTML can be bad practice, but for prototyping it's all good fun!

Key Examples

This "site" directory illustratrates the use of many of the key controls in the AribaWeb "Widgets" framework:

Master Detail App (ER Editor): A mini app illustrating data tables, tab sets, master-detail selection, client panels, ...

Data Table Samples: A set of examples showing many facets of the DataTable widget.

Pivot Table Samples: Examples of the Pivot Table support in AWTDataTable, including outline tables, and tables with rich mixtures of dimensioned and non-dimensioned data.
